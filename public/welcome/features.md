# Features Overview

Discover all the powerful features that make Learning Notes the perfect platform for your documentation needs.

## 📝 **Content Creation & Editing**

### **Rich Markdown Support**
- GitHub Flavored Markdown (GFM)
- Syntax highlighting for 100+ programming languages
- Mathematical equations with LaTeX support
- Tables, lists, and task lists
- Custom HTML components

### **Live Editor**
```markdown
# Example Code Block
const greeting = (name: string) => {
  return `Hello, ${name}!`
}
```

### **Inline Editing**
- Click any file and hit the edit button
- Real-time preview as you type
- Auto-save functionality
- Undo/redo support

## 🔍 **Search & Discovery**

### **Powerful Search Engine**
- **Full-text search** across all files
- **Real-time results** with instant feedback
- **Context highlighting** shows relevant snippets
- **Keyboard shortcuts** for quick access

### **Smart File Discovery**
- Automatic detection of markdown files
- Hierarchical folder structure
- Recent files and bookmarks
- Tag-based organization
